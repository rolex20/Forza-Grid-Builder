# [Forza Grid Builder](https://rolex20.github.io/Forza-Grid-Builder/Forza-Grid-Builder.html)

- A Forza Grid Builder app for building event restrictions and matches from car datasets (FM‑2023, FM7, and more).

- If you live for clean laps and tighter event rules, this app is for you — it lets you build Forza event restrictions and matches from curated car datasets in a fast, single‑page experience. To use this app, simply browse [here](https://rolex20.github.io/Forza-Grid-Builder/Forza-Grid-Builder.html) and drop it into your favorites so it’s always one click away when inspiration hits.

- **Tech stack highlights:** HTML5, CSS3 (custom properties, responsive layouts), vanilla JavaScript (ES6+), DOM-driven UI rendering, dynamic filtering/search, localStorage state persistence, dataset normalization, and pure client‑side performance tuning with zero external dependencies.
